# Pyproject
PROJECT TITLE : Pyproject

OVERVIEW OF THE PROJECT : This project is a two-player “Tic-Tac-Toe game” built using Python’s Tkinter library for  the graphical user interface. The program allows two players to play alternately by clicking on buttons arranged in a 3*3 grid. This application checks for winning  combinations after every move, highlights the winning cells in green and displays the winner through a popup message. It provides an engaging way to understand python GUI programming, event handling and logic implementation.

FEATURES :
Interactive GUI
Winner Detection
Player Switching
Popup Notification
Logical Game Rules

TECHNOLOGIES/ TOOLS USED :
 Python programming language has used.
 Tkinter(GUI Library)
 It is developed in IDLE
