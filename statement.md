# PROBLEM STATEMENT:
Learning programming concepts often becomes theoretical and lacks practical engagement. Begineers often struggle to apply logical thinking and user interface desgin skills in real-world projects. This project aims to solve that by developing an interactive Tic-Tac-Toe game using Pyhton's tkinter library.
# SCOPE OF THE PROJECT:
The project focuses on creating a two-player Tic-Tac-Toe game that runs on any computer with python installed. The scope can be expanded in the future to include features like automatic reset or replay options, score tracking, improved interface with animations
# TARGET USERS :
Beginners and students in python programming - to learn GUI and logic buliding interactively.
Educators. trainers - to use the project as a teaching example for GUI programming.
Therefore, any anybody who are interested in playing desktop-based game for fun.
# HIGH LEVEL FEATURES:
Built using Tkinter for easy and interactive gameplay.
Automatically alternates turns between player X and O.
It has win detection.
Displays message boxes to announce the winner.
It has colour highlighting features.
